File Permissions

Change mode of file to 775

folder
Recursively chmod folder to 600
chown
user :group file
Change file owner to user and
group to group
File Permission Numbers
The first digit is the owner permission, the second
the group and the third for everyone.
Calculate each of the three permission digits by
adding the numeric values of the permissions below.
4
read (r)
2
write (w)
1
execute (x)
