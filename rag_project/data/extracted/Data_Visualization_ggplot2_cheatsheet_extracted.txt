Allow scales to roam free (scales)
The default for multi-panel plots in ggplot2 is to use equivalent scales in each panel. But sometimes you want
to allow a panel's own data to determine the scale. This is not often a good idea since it may give your user the
wrong impression about the data but to do this you can set scales="free" like this:
ggplot(nmmaps, aes(date,temp))+geom_point(color="chartreuse4")+
  facet_wrap(~year, ncol=2, scales="free")
