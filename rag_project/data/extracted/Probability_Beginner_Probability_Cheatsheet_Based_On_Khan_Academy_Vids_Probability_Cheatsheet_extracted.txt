Overview
Probability provides measures for reasoning the like-
lihood that an event will occur in an experiment.
2
Terminology
1. Experiment: Procedure that yields one of the
possible outcomes. E.g. Tossing a coin.
2. Sample Space S: A set of all possible outcomes
of an experiment. E.g. S = {Head, Tail} when
tossing a fair coin.
3. Event E: Set of outcomes of an experiment.
Note that if the Event contains only one out-
come, it is then called an elementary event. E.g.
The event that coin is Head.
3
Random Variable R.V
A variable that its possible values are the outcomes
of an experiment. It has two types:
1. Discrete R.V: Take one countable number of
distinct values. E.g 1, 2, 3, 4
2. Continuous R.V: Take inﬁnite number of pos-
sible values (usually measurements).
E.g.
all
interval values in the rage [1 - 2].

P(B).
(b) Dependent Events: P(A ∩B) = P(A) ×
P(B|A). where P(B|A) read: Probability
of B given A had happened.
