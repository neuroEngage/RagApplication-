A Quick Side Note About Color
Colorspaces:
●
RGBa - (red, green, blue, alpha) usually 8bits per channel
●
Hexadecimal - 2 hex values per R, G and B (#00FF00)
●
CMYK - Cyan, Magenta, Yellow, Black
●
HSV - Hue, Saturation, Value

Interactive Plotting
Plotly
A python package written by Hadley Wickham that uses D3 
under the hood
●
Making interactive plots less cumbersome
●
Input arguments look like JSON objects
●
Documentation is hard to read and navigate

--- PAGE 39 ---
Interactive Plotting
Plotly
They are trying to run a business, so they imply that you need 
an account to make plots...Thankfully this isn’t true. Just run it 
in `offline` mode.

--- PAGE 47 ---
Interactive Plotting
Folium
Allows us to use Python to interact with Leaflet.JS mapping 
tools much in the same why Mapbox and Plotly do, but 
without any accounts or access-tokens.
